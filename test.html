dim_products.csv

Product catalog with 500 SKUs including physical attributes, demand patterns, and inventory planning parameters.

Column Name Data Type Description Sample Values
SKU PK String Stock keeping unit identifier SKU0001, SKU0234, SKU0500
Product_Name String Full product name Aanya Silk Anarkali, Nova Leather Loafers
Category String Primary product category Dress, Shoes, Accessories
Sub_Category String Detailed product subcategory Silk Anarkali, Leather Loafers, Silver Jhumkas
Weight_kg Float Product weight in kilograms 0.3 - 1.5 (varies by category)
Volume_cbm Float Product volume in cubic meters 0.001 - 0.020 (varies by category)
Base_Demand_Min Integer Minimum daily demand baseline 3 - 25 (varies by category)
Base_Demand_Max Integer Maximum daily demand baseline 10 - 25 (varies by category)
Avg_Daily_Demand Float Calculated average daily demand (Min + Max) / 2 * 0.80
ROP Integer Reorder point (safety stock) Calculated: Avg_Demand * LT_Mean + Safety_Stock
Target_Level Integer Target inventory level ROP + (7 days * Avg_Daily_Demand)

Relationships

  • Referenced by fact_orders.SKU
  • Referenced by fact_inbound_shipments.SKU
  • Referenced by fact_inventory_snapshot.SKU
  • Referenced by fact_returns.SKU
